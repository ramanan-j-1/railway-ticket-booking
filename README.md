# Railway Ticket Booking System

A Java-based Railway Ticket Booking System that simulates an online platform for booking train tickets. This project allows users to reserve tickets, check seat availability, calculate fares, and manage waitlists for different travel classes.

---

## Features

- Passenger ticket booking
- Multiple travel classes:
  - Sleeper
  - AC
  - First Class
- Seat availability tracking
- Automatic fare calculation
- Senior citizen discount
- Free tickets for children below 5 years
- Waitlist management when seats are unavailable
- Booking confirmation with ticket details

---

## Technologies Used

- Java
- Scanner Class
- ArrayList
- Conditional Statements
- Loops
- Switch Case

---

## Project Structure

- User enters passenger details
- Selects travel class
- Chooses number of tickets
- System checks seat availability
- Fare is calculated
- Ticket is generated

## How to Run

1. Clone the repository:
```bash
git clone <your-repository-link>
```

2. Open in your Java IDE or terminal.

3. Compile the program:
```bash
javac RailwayTicketBooking.java
```

4. Run the program:
```bash
java RailwayTicketBooking
```
